# MyScope Admin Panel Setup

## Install Required Dependencies

Run the following command to install all required packages:

```bash
npm install tailwindcss-animate class-variance-authority clsx tailwind-merge lucide-react
```

## Theme Configuration

The admin panel is configured with:
- **Dark Mode**: Default theme (can toggle to light mode)
- **Primary Color**: Emerald (#10B981)
- **Secondary Color**: Indigo (#6366F1)
- **Accent Color**: Pink (#F472B6)

## Features

- ✅ Dark/Light mode toggle
- ✅ Consistent color scheme matching MyScope brand
- ✅ Tailwind CSS with shadcn/ui compatibility
- ✅ Custom scrollbar styling
- ✅ Focus states and accessibility

## Next Steps

1. Install dependencies (see command above)
2. Run `npm run dev` to start the development server
3. Access the admin panel at http://localhost:3000
